Node.js - это кросс-платформенная, открытая серверная среда, которая может работать на Windows, Linux, Unix, macOS и многих других. Node.js - это среда выполнения JavaScript на стороне сервера, работает на движке JavaScript V8 и выполняет код JavaScript вне веб-браузера.

Node.js позволяет разработчикам использовать JavaScript для написания командных строк и для серверного скриптинга. Возможность запускать код JavaScript на сервере часто используется для генерации динамического содержимого веб-страницы до того, как страница отправляется в веб-браузер пользователя. Следовательно, Node.js представляет собой парадигму "JavaScript везде",[6] объединяющую разработку веб-приложений вокруг одного языка программирования, вместо использования разных языков для программирования на стороне сервера и клиента.

Node.js имеет архитектуру, основанную на событиях, способную на асинхронный ввод/вывод. Эти дизайнерские решения направлены на оптимизацию пропускной способности и масштабируемости в веб-приложениях с большим количеством операций ввода/вывода, а также для реальных веб-приложений (например, программы реального времени и браузерные игры).[7]

Проект распределенной разработки Node.js ранее управлялся Фондом Node.js,[8] и теперь объединился с Фондом JS для создания Фонда OpenJS. Фонд OpenJS облегчается программой Совместных проектов Фонда Linux.[9]
